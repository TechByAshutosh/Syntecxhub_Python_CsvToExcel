# CSV to Excel Converter

## Week 3 - Project 1
### By SyntexcHub

A simple Python automation project that converts CSV files into Excel (`.xlsx`) format using Pandas and OpenPyXL.

This project also performs:
- Column name cleaning
- Missing value handling
- Automatic output folder creation
- Basic error handling
